# 学生宿舍管理系统 java209

## 项目概述

学生宿舍管理系统是基于Springboot和Vue的前后端分离项目，采用MVC三层架构，结合Spring、SpringBoot、Mybatis和Vue技术栈进行开发。系统使用MySQL作为数据库存储，实现了浏览器/服务器架构（B/S）的Web应用。该系统界面简洁，操作方便。

## 技术栈

- 前端：Vue
- 后端：Springboot、Javaee
- 数据库：MySQL
- 构建工具：Maven

## 功能模块

- 学生信息管理
- 宿舍楼信息管理
- 宿舍分配管理
- 维修申请管理
- 缴费信息管理

## 系统角色

- 管理员
- 学生

## 运行环境

- 操作系统：兼容Windows、Linux和macOS
- 数据库：MySQL 5.7 或更高版本
- Java环境：JDK 1.8 或更高版本
- Web服务器：内置Tomcat

## 部署步骤

1. 准备MySQL数据库环境并创建相应数据库
2. 导入项目提供的数据库文件
3. 使用Maven构建项目，获取最新的依赖包
4. 部署后端服务，确保Web服务器运行正常
5. 部署前端Vue项目，确保可访问后端API

## 目录结构

```
- backend                    // 后端代码目录
  - src
    - main
      - java
        - com.example.dormitory     // 主要代码包
- frontend                   // 前端代码目录
  - src
    - assets                     // 静态资源
    - components                  // Vue组件
    - views                       // 页面视图
- pom.xml                    // Maven项目文件
```

- 前后端分离，易于管理和维护
- 使用Springboot，提高开发效率
